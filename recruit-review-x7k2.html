② 자소서가 러너블스러운가 서비스·러닝·사용자 맥락을 한 번이라도 자기 언어로 연결했는가.
③ 브랜드 마케터 개념 이해 "스포츠 마케터=IP 기획운영"이 아닌 "이걸 팔 가능성 있는 사람". B2C 브랜드·판매 관점.
④ 연봉 3천대 수용 구조적 업턴 없이 이동 가능한 레벨. 3,500+ 경력자는 이탈 리스크.

A. 기존 탈락자 비교 포인트 WHY

통과자 3명 vs 내가 Top에 올렸던 탈락자

왜 점수가 높았던 신진·박유진·김예은이 떨어지고, 중간 점수였던 김소연이 통과했는가 — 새 기준 4축으로 역추적

① 포트폴리오 마케팅 포폴 제출 (부트캠프 실험 기록) 마케팅 포폴 제출 (JYP·에이전시 캠페인) PPT 포폴 제출 (광고동아리 수상) ✗ 이력서만 ✗ 이력서만 개인 포폴 웹
② 러너블스러운 자소서 "러너블 앱 직접 설치, 초보 러너의 막막함" 명시 IP·아티스트 마케팅 본질 언급 "소비자 인사이트를 경험으로 연결" ✗ 비영리 PR 총괄 일반론 ✗ "스포츠 마케터" 반복, 브랜드 관점 부재 ✗ 영화 IP 경력 나열, 러너블 연결 없음

④ 연봉 3천대 신입 ✓ 직전 3,870 (보더라인) 희망 3,200~3,400 ✓ 경력 1y, 추정 3천대 신입 ✓ ✗ 희망 5,000~6,000

🔑 v1 분석의 오류 자기교정

  • 신진 (90점 → 탈락): "아디다스 러닝 캠페인 PM"이라는 단어만 보고 JD 1:1 매칭이라 판단했으나, 실제로는 비영리 기관 PR·언론홍보가 본질. 러너블이 원하는 "이걸 팔 수 있는 사람"이 아니라 "캠페인 운영자". 자소서에 러너블 맥락도 없음.
  • 박유진 (80점 → 탈락): 점수 축에서 "러닝/스포츠 경험"에 25점을 몰아줬으나, 자소서 전반의 "스포츠 마케터" 지향 = IP 기획운영 유형이었다. 러너블은 이 사람들을 이미 가지고 있고, 필요한 건 이걸 파는 사람. 포폴 부재도 리더 시그널 감점.
  • 김예은 (71점 → 탈락): IP 마케팅 4년차 경력 훌륭하나 희망 연봉 5-6천이 단순 "협의" 수준이 아닌 구조적 미스매치였음.
  • 김소연 (66점 → 통과): v1에서 "영양사 배경"이라 B로 깎았으나, 자소서의 "러너블 앱 직접 설치→초보 러너 관점" 한 문장이 리더 시그널 #2(윗사람 의도 파악)의 가장 강력한 증거였음.

교훈: "실적 키워드 매칭" 방식은 "이 사람이 러너블에서 어떻게 움직일지"를 예측하지 못한다. 리더 시그널 4축 (포폴 / 자소서 러너블 맥락 / 파는 사람 여부 / 연봉 레벨)이 더 정확한 스크리닝 축이다.

  • 🔥 건축 PPT 템플릿 누적 판매 456건, 목표 대비 400%+

판정: 박가영 단독이 1순위. UX 결이 팀에 필요할 때만 면접 1회 추가 가치.

🔴 오세현 — v1 A- 하향 → 탈락

v1에서는 "콘텐츠 IP 풀스택 = 이승미와 유사"로 A-를 줬으나, 새 기준 4축으로 재평가하면 3가지 리스크가 동시에 존재

  1. 연봉 구조적 미스매치 — 직전 연봉 3,800. 3천대 초반 딜하면 구조적 하락 = 이탈 리스크. 이승미(3,870) 통과는 예외 케이스였을 가능성.
  2. 자소서 톤이 팔로워 — 마지막 단락 "밝고 긍정적인 성격... 최선을 다하겠습니다... 팀 분위기를 살리는 데 기여" = 분위기 메이커 팔로워 색채. 통과 3명 자소서 구조("문제 정의 → 가설 → 실행 → 회고")와 다름.
  3. 러너블 맥락 전무 — 자소서에 러너블·러닝·서비스 언급 0. "콘텐츠의 본질은 공감과 확산" 같은 일반론. 박가영(전용 포폴) / 김소연(앱 직접 사용) 대비 러너블스러움 부재.

판정: 이승미가 있으므로 경력+IP 축은 이미 확보됨. 오세현 추가는 라인업 균형에도 불필요.

💡 4인 라인업의 전략적 의미

김소연 + 박가영 (신입 2명) = 서로 다른 각도에서 "러너블을 이해한다"는 시그널

  • 김소연: 러너블 앱 직접 사용자 관점 (관찰자)
  • 박가영: 러너블 전용 포폴 + 판매 실행자 (실행자)
  • 이승미: 경력 IP·에이전시 풀스택 (파는 경력자)
  • 이호범: 스포츠 IP 팝업 현장 PM (판매 경험자)
